TeaNode

Click the toolbar button to toggle the floating overlay.

Getting started

If you see a red ! badge on the extension icon, the relay server is not reachable. Start the TeaNode node on this machine, then click the toolbar button again.

Relay URL

Default: http://127.0.0.1:8833. The URL of your TeaNode node.

Authentication Token

Bearer token for authenticating with the TeaNode node. Create one in Settings → Tokens on the web UI.
