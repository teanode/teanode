# Skills

Learned procedures, workflows, and know-how. Document reusable skills here so you can apply them in future conversations.

## How to Use This File

- When you figure out a multi-step procedure, write it down here as a skill
- When a user teaches you a workflow or technique, document it
- Organize skills with clear headings so you can find them later
- Keep each skill concise: name, when to use it, and step-by-step instructions
- When multiple skills might apply, pick the most specific one first
- Prefer a one-skill-first approach; combine multiple skills only when the first one is not enough
